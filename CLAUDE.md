# Claude Code briefing

This repo is the **Rust on Zephyr** workshop for RustWeek 2026 (Utrecht, 2026-05-18). The scaffold pass is complete; per-exercise content work is driven by [`TASKS.md`](TASKS.md) at the repo root. **Read `TASKS.md` first** — it is the spec for any exercise-content pass.

Only flesh out exercises marked **READY FOR CODE PASS** in `TASKS.md`. Entries marked **DRAFT UNSPEC** are still being designed in the planning conversation; leave them alone.

The shared-constraints section at the top of `TASKS.md` (Apache-2.0 headers, T2 topology, canonical build invocation from inside the exercise dir, `Cargo.lock` commit policy, symlink trick referenced from `common/README.md`) applies to every file you create unless an exercise explicitly overrides it.

For the original scaffold-pass brief that produced this skeleton, see the archived `repo-scaffold.instructions.md` in the presenter's notes folder. It has known drift from the as-built repo and is kept only as historical record.
